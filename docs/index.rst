Welcome to Django WalletOne's documentation!
============================================

Django WalletOne is a pluggable application that integrates
with WalletOne online payments provider.

Before any using this application it is strongly recommended
to go throug the `official Wallet One documentation <https://www.walletone.com/en/merchant/documentation/>`_.
